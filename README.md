# Trabajo Final Integrador

Sistema de Inscripcion a Cursos Academicos.

## Descripcion

Este repositorio contiene el trabajo final integrador para la materia, con separacion de responsabilidades entre:

- Backend: API y logica del servidor (Node.js + Express + PostgreSQL).
- Frontend: interfaz web del sistema.
- Consigna: documentos, diagrama y script SQL base.

## Estructura real del proyecto

```text
progIV/
├── README.md
└── tpintegrador/
	├── backend/
	│   ├── package.json
	│   ├── README.md
	│   └── src/
```

## Alcance funcional

El sistema apunta a cubrir:

- Login de usuario.
- Visualizacion general en dashboard.
- Gestion de estudiantes.
- Gestion de cursos.
- Gestion de inscripciones.

## Tecnologias

- Backend: Node.js, Express, PostgreSQL.
- Frontend: HTML y CSS.
- Base de datos: SQL en `tpintegrador/consigna/scripts.sql`.
