# 📚 SICA

## Sistema de Inscripción a Cursos Académicos

Sistema desarrollado para la gestión e inscripción a cursos de la Facultad de Ciencias de la Administración.

## Requisitos

- `NodeJs`
- `PostgreSQL`
- `NPM`

## ⚙️ Instalación

1. Clonar el repositorio:

```bash
git clone https://github.com/valenmar01/progIV.git
cd sica
code .
```

2. Crear archivo `.env` (opcional por ahora):

```env
PORT=3000
```

### Instalar dependencias

```bash
npm install
```

## Levantar el servidor en modo prueba

```bash
npm run dev
```
